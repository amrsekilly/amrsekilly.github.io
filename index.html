AquaPhoton.net Responsive Website

I used Handlebars.js to create the templates, to separate the content in a JSON file from the HTML, so that it's easy for anyone to edit.

I used LESS to write my CSS, it helped my write an organized and clear styles. And I used Grunt, and bower for managing the needed libraries and doing the redundant tasks.\n\nI also used Bootstrap for responsiveness, and I used Javascript and some JS libraries such as jQuery for DOM manipulation and wow.js for creating landing elements when content is revealed by scrolling.

Also I used animate.css to add some animations to the DOM elements.

I created an image viewer from scratch using jQuery, and I implemented a random generator, for generating a new array for the team members on each refresh, so that each member appears in a new position on the screen on each refresh.
